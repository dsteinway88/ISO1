What is the Bit9 Security Platform?

The Bit9 Security Platform is a policy-driven application control and whitelisting solution that protects Dechert employees from modern security threats by detecting and preventing attacks against the firm's laptops and desktops.

Bit9 has been integrated with the firm's security monitoring and malware detection systems, and provides a forensic assessment of where and when an attack originated and real-time visibility into suspicious file and process activity across all desktop and laptop computers.

In short, the whitelisting technology allows end-users to install and run legitimate software and devices while also providing a way to prohibit anything unauthorized or known to be malicious from executing.
